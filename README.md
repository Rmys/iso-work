iso-work
========

Pisi Linux iso creation tools and files

iso work yalı ile çalışan pisi iso oluşturma aracıdır. çalıştırmadan önce aşağıda listesi olan paketlerin sisteminizde olmasına dikkat edin.

Kurulması gereken paketler:
===========================
- cdrkit
- syslinux
- libisoburn
- python-sip
- squashfs-tools
- pyqtermwidget5 (Required for GUI)
- python-requests
- python-chardet
- python-urllib3
- python-idna
